STARS BILLING - Aplikasi Billing Mikrotik Berbasis Web Untuk Hotspot dan PPPoE



STARS BILLING - adalah aplikasi berbasis web untuk mempermudah manajemen wifi hotspot menggunakan router mikrotik, khususnya router mikrotik yang belum mendukung usermanager.

Kami menggunakan RouterOS API Class PHP untuk dapat terhubung ke router anda melaui API Service Mikrotik.

Aplikasi ini dapat anda gunakan menggunakan browser pada desktop atau smartphone anda dan tidak perlu mengunduh dan memasang aplikasi tambahan pada perangkat yang anda gunakan karena stars billing dipublikasikan secara online dan aktif 24 jam pada server kami.

Untuk menggunakannya anda harus menggunakan IP Public atau VPN Remote yang terhubung ke router mikrotik anda melalui port API (8728)

Fitur yang tersedia saat ini yaitu :

  • Manajemen sales voucher, Untuk mempermudah dalam menghitung total penjualan agen voucher anda secara otomatis.
  • Manajemen laporan penjualan voucher per sales atau agen penjualan voucher
  • Manajemen pembayaran hasil penjualan voucher
  • Manajemen PPPoE profile, Untuk mempermudah proses pembuatan limitasi bandwidth dan masa aktif pengguna
  • Tambah, Edit dan Hapus PPPoE user, Untuk mempermudah proses pengelolaan username, password dan profile user PPPoE
  • Isolir Expired PPPoE User, Untuk meanonaktifkan koneksi internet pada pengguna yang belum melakukan pembayaran.
  • Monitoring Active PPPoE User, Untuk memantau daftar pengguna yang sedang aktif.
  • Manajemen Profile Hotspot, Untuk mempermudah proses pembuatan limitasi bandwidth dan masa aktif voucher WiFi
  • Manajemen Hotspot User, Untuk mempermudah pengelolaan kode voucher WiFi
  • Generate Voucher, Untuk mempermudah proses pembuatan kode voucher
  • Print Voucher, Dengan beberapa pilihan template (Small, Normal, QR Code) dan sudah mendukung Thermal Printer
  • Mudah digunakan, Dengan design responsive dan user friendly. Kami menggunakan AdminLTE agar tampilan aplikasinya lebih keren dan nyaman saat digunakan.
  • Akses Online sehingga dapat anda gunakan kapanpun dan dimanapun sesuai kebutuhan anda melalui jaringan internet
