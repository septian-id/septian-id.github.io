STARS BILLING

Aplikasi Manajemen WiFi Hotspot Berbasis WEB

STARS BILLING V4

Release Date, November 7, 2023

STARS BILLING adalah aplikasi berbasis web untuk membantu manajemen Hotspot MikroTik.

Stars Billing mempunyai beberapa fitur yang sangat berguna yang akan memudahkan anda dalam memanajemen pelanggan internet anda dan penjualan voucher hotspot.

Dengan aplikasi ini anda tidak perlu lagi repot membuat catatan penjualan voucher secara manual untuk mengetahui berapa total penjualan masing-masing agen voucher anda secara realtime.

Ini dapat anda gunakan secara offline terinstall langsung di laptop / PC anda dan juga dapat digunakan secara online terinstall di web hosting.

Fitur yang tersedia adalah :

  • Sales Member -> Daftar agen penjualan voucher.
  • Sales Detail -> Detail penjualan voucher masing-masing agen voucher.
  • Reporting -> Filter data penjualan voucher berdasarkan periode tertentu.
  • Profile -> Pengelolaan profile WiFi hotspot.
  • Sales Member -> Daftar agen penjualan voucher.
  • PPPoE Profile -> Pengelolaan profile PPPoE, Tersedia 2 mode expire (disable & redirect) setelah masa aktif habis.
  • PPPoE Secret -> Pengelolaan daftar user PPPoE
  • PPPoE Active -> Monitoring status dan penggunaan data user PPPoE yang sedang aktif.
  • Hotspot Users -> Pengelolaan voucher WiFi yang terdaftar di mikrotik.
  • Hotspot Active Users -> Monitoring user hostspot yang sedang aktif.
  • Hotspot IP Bindings -> Bypass / Blocking MAC Addrees tertentu.
  • Hotspot Check Status -> Pengecekan status kode voucher tertentu apakah statusnya belum digunakan, Sudah digunakan tapi sedang offline, Sudah Kadaluarsa atau Sedang aktif digunakan.
  • Hotspot Generate -> Generate kode voucher dengan jumlah yang banyak.
  • DHCP Leases -> Monitoring perangkat yang terhubung ke jaringan.
  • System Scheduler -> Monitoring masa aktif dan durasi voucher yang sedang digunakan.
  • Template Editor -> Editor HTML untuk mempermudah membuat design voucher WiFi hotspot anda.
