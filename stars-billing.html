Stars Billing - Aplikasi Billing Hotspot Mikrotik Berbasis Web

STARS-BILLING - adalah aplikasi berbasis web untuk membantu manajemen Hotspot MikroTik. Khususnya MikroTik yang tidak mendukung User Manager.

Aplikasi billing ini dilengkapi dengan :

  • Manajemen Reseller / Sales Voucher
  • Manajemen PPPoE User
  • Manajemen Hotspot Voucher.
  • Printing Voucher (Small, Normal, QR Code dan Mendukung Thermal Printer)
  • Monitoring Hotspot Active
  • Monitoring PPPoE Active
  • Detail Laporan Penjualan Voucher Per Sales Voucher
  • Dan masih banyak lagi..

Billing ini dipublikasikan secara online agar dapat anda gunakan kapanpun dan dimanapun melalui jaringan internet sehingga anda tidak perlu repot mengunduh file dan melakukan installasinya sendiri.

Jika kalian tertarik untuk mencobanya, Kalian bisa mendapatkan uji coba gratis 3 hari sebelum berlangganan.

Untuk melakukan pendaftaran akun, Silahkan klik disini : https://starsbilling.ddns.net/signup.php

Jika sudah punya akun silahkan Log In disini : https://starsbilling.ddns.net/login.php
